## 🚀 Overview

**LastMinute AI** is an advanced, aesthetically stunning "Smart Workspace" designed to rescue users from overwhelming deadlines. It features a spatial 3D interface, intelligent task extraction, workload visualization, and deep-focus modes.

The platform is divided into 5 Core Subsystems:

1. **Plan Console**: Dump your brain via voice or text. The system parses unstructured data into prioritized tasks with calculated risk scores and estimated durations.
2. **Timeline View**: A beautiful alternating timeline mapping out your entire task queue with categorized color-coding and floating ambient elements.
3. **AI Subsystems (Nexus & Aegis)**: 
   - **Nexus (Executor)**: Generates actionable checklists and step-by-step subtasks.
   - **Aegis (Risk Analyzer)**: Calculates the threat percentage of missing deadlines based on task urgency.
4. **Workload Insights**: A dynamic, interactive bar chart visualizing your cognitive load and energy distribution.
5. **Focus HUD Room**: A distraction-free environment featuring a built-in Lo-Fi Radio player, animated visualizer, daily motivational quotes, and a task completion celebration system.

---

## ✨ Key Features

- **Spatial Design System**: Uses glassmorphism, floating orbs, bento grids, and 3D hover effects.
- **Micro-Animations**: Powered by `framer-motion` for buttery smooth transitions, expanding panels, and spring animations.
- **Smart Task Extraction**: Automatically deduces urgency, risk of failure, and priority levels from plain text.
- **Lo-Fi Focus Radio**: Built-in audio player with continuous ambient tracks to keep you in the zone.
- **Secure Local Authentication**: AES-256 local verification simulation with sleek sign-in/register flows.

## 💻 Tech Stack

- **Frontend**: React (v18), TypeScript
- **Styling**: Tailwind CSS + Custom CSS (`index.css`)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## 🛠️ Local Development

To run LastMinute AI locally:

1. Clone the repository
   ```bash
   git clone https://github.com/raaneshkv/lastminute.git
   ```
2. Navigate into the project directory
   ```bash
   cd lastminute
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm run dev
   ```

*You can sign in using any email and the password `demo123`, or register a new local session.*
